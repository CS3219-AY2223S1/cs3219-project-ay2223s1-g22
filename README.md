# For developers

To set up the environment for local development, run the following commands in SEPARATE TERMINALS:

#### Initialize API Gateway

```
cd api-gateway
npm run dev
```

#### Initialize Matching Service

```
cd matching-service
npm run dev
```

#### Initialize User Service

```
cd user-service
npm run dev
```

#### Initialize Frontend

```
cd frontend
npm run start
```
